Ficheiro de Ajuda da Linha de Comandos do MCOS!

Comandos: 
	@help -> mostra uma ajuda resumida dos comandos
	@info -> mostra a versao do sistema operativo
	@cls  -> apaga o ecran
	@ls   -> lista os ficheiros da drive
	@type -> mostra o Conteudo de um ficheiro
	@exit -> faz um halt ao sistema
	
Teclas de macro:
	F1 -> @help
	F2 -> @info
	F3 -> @cls
	F4 -> @exit
	F5 -> @ls

Para que os dados sejam correctamente guardados
fazer sempre o halt ao sistema, para terminar a sessao!

O sistema Operativo encontra-se em estado de desenvolvimento.

Objectivos a curto prazo:
	instalar sistema de device drivers

Objectivos a medio prazo:
	suporte para subdirectorias
